## Overview
This repository contains a presentation-based UX/UI redesign of the Savana landing page, focused on improving the desktop user experience.

The goal of this project was to analyze usability issues, restructure navigation, and present a clearer visual hierarchy through a conceptual redesign
---

## Problem Statement
The existing interface feels mobile-first on desktop, with dense navigation and promotional overload causing cognitive strain.

---
## Scope of Work
- Desktop-first landing page redesign
- UX-focused analysis and information architecture
- UI execution for presentation purposes
- No functional implementation or live deployment

---
## UX Goals
- Reduce cognitive load
- Simplify navigation
- Improve product discoverability
- Maintain brand identity

---

## Process
1. UX analysis of existing landing page
2. Navigation restructuring (5 primary categories)
3. Lo-fi wireframing
4. UI redesign for presentation
5. High-fidelity UI design

---

## Deliverables
- Lo-fi wireframes
- Desktop landing page redesign
- UX documentation
- Presentation slides

---

## Note
This project is a conceptual redesign intended to demonstrate UX thinking and UI execution, not a fully functional website.
